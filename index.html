ใส่ Project URL และ Publishable/Anon Key จาก Supabase แล้วกดเชื่อมต่อ

ขั้นตอนสร้างตาราง

  1. เปิด Supabase → SQL Editor
  2. เปิดไฟล์ supabase_schema.sql ที่อยู่ในโฟลเดอร์นี้
  3. คัดลอก SQL ทั้งหมด → Run
  4. กลับมาหน้านี้ ใส่ URL และ Key
